#### Descrição
Este projeto tem como objetivo analisar dados de vendas de uma empresa de varejo, considerando informações de clientes, produtos e comportamento de compra.

---

#### Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Datetime

---

#### Etapas do projeto

##### 1. Carregamento de dados
Leitura de arquivos CSV
Verificação inicial da base

--
##### 2. Limpeza de dados
Tratamento de valores nulos (NaN, # N/D)
Padronização de textos (maiúsculo, remoção de espaços)
Conversão de datas para formato datetime

--
##### 3. Transformação de dados
Agrupamentos com groupby()
Criação de tabelas analíticas

---

#### Principais insights
- Alto volume de compras de fralda por clientes sem filhos
- Concentração de vendas em poucas categorias
- Diferenças de consumo entre classes sociais

---

#### Problemas encontrados

- Dados faltantes (`NaN`, `#N/D`)
- Inconsistência na quantidade de filhos
- Falta de padronização em nomes de produtos
